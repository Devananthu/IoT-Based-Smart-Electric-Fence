# FenceIQ – Smart Electric Fence Monitoring System

## Overview
FenceIQ is an IoT-enabled real-time electric fence monitoring system designed for intelligent perimeter security and live fence status analysis. The system integrates embedded hardware, sensor-based monitoring, and a modern web dashboard to detect fence abnormalities such as intrusion attempts, line tampering, fence cuts, and fault conditions.

The project combines embedded systems, IoT communication, and real-time visualization to provide continuous monitoring and instant alert generation for secured environments.


## Features

- Real-time electric fence monitoring
- Intrusion and tampering detection
- Live dashboard with dynamic status updates
- Sensor-based fault analysis
- IoT-enabled remote monitoring
- Real-time alert and warning system
- Interactive graphical interface
- Embedded hardware and web integration
- Scalable architecture for industrial applications

## Project Workflow

1. The electric fence is continuously monitored using ESP32-based sensing circuitry.
2. Sensor data is processed in real time to detect abnormal fence conditions.
3. The microcontroller transmits live monitoring data through IoT communication.
4. The web dashboard receives and visualizes live fence status updates.
5. Alerts are triggered instantly when intrusion, tampering, or fault conditions are detected.
